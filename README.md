# Contact-Book-in-C-
This is a simple Contact Book program written in C++ that allows a user to manage contacts. The program demonstrates the use of classes, arrays, global variables, and basic input/output operations. Users can add, display, search, and delete contacts.

     Its Features
Add multiple contacts without specifying the number in advance
Display all contacts with name and phone number
Search for a contact by name
Delete a contact by name
Handles multiple contacts using a fixed-size array and a global counter

     And It Uses
C++
Standard Input/Output (cin, cout)
Classes and objects
Arrays
Global variables
